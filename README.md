# Lecture_Evaluation
3주에 걸쳐 제작하게 된 강의평가의 README 파일입니다.

<img width="700" src="https://user-images.githubusercontent.com/62650222/79684774-ed736e80-826e-11ea-9989-34d451ec8759.PNG">



- ### 강의평가 취지

>1.  유튜브에서 강의를 접해 클론 형식으로 제작했습니다.
>2.  프론트/백 두가지 영역을 함께 배울 수 있었기 때문에 프로젝트를 진행했습니다.
>3.  혼자 진행하기 때문에 개발셋팅, 코드 오류 과정에서 시간을 꽤 보냈지만,
     이것 또한 저에게는 유익했고, 전반적인 웹 프로세스 과정을 배우게 되었습니다!:slightly_smiling_face:

- ### 사용한 기술

>1.   개발환경 : Chrome, JDK, Eclipse
>2.   서버 : Tomcat
>3.   라이브러리 : Bootstrap, Popper, Jquery
>4.   데이터베이스 : MySQL

- ### 세부기능 설명

>1.   로그인
\- 평가를 위해선, 가장 먼저 로그인이 필수적입니다. 로그인을 하지못하면 수강평가를 진행할 수 없습니다.

<img width="600" src="https://user-images.githubusercontent.com/62650222/79685007-76d77080-8270-11ea-9589-474d8cc48770.PNG">

>2.   회원가입
\- 회원가입을 할땐 반드시 이메일 인증으로 본인확인 과정을 거쳐야 합니다.

<img width="600" src="https://user-images.githubusercontent.com/62650222/79685148-565be600-8271-11ea-9cb9-cdb2306f41ff.PNG">

>3.   이메일 전송
\- 전송받은 링크를 누르면, 자동적으로 가입이 완성됩니다. 메일 확인하지 않고 로그인을 하게되면 메일 재전송이 가능합니다.


<img width="600" src="https://user-images.githubusercontent.com/62650222/79685186-a3d85300-8271-11ea-8790-2092a9a1c25b.PNG">

이메일 인증 확인을 하지 않았을때 재전송이 가능합니다.

<img width="600" src="https://user-images.githubusercontent.com/62650222/79685229-fd408200-8271-11ea-91ce-9f37e614731b.PNG">
